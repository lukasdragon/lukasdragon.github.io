AquaConsole

AquaConsole is an open-source Microsoft CMD inspired application
Which I had designed and programmed to teach myself C# around the age of 14.

Open-Source

Like all my software, AquaConsole is open-source and licensed under The GNU General Public license! This means that you may modify and/or use it however you like with just a few conditions. Under the GNU You may copy, distribute and modify the software as long as you track changes/dates in source files. Any modifications to or software including (via compiler) GPL-licensed code must also be made available under the GPL along with build & install instructions.

  • Source-Code

Features

  • A robust plugin system allowing the user to download and install third-party plugins from within the application.
  • A modular command system, allowing developers to develop their own plugins efficiently and with ease.
  • A wide range of commands of which are familiar from Command Prompt that are already in the program or that will be added soon.
